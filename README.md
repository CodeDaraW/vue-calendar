# vue-calendar

> date-picker component for Vue.js [demo](http://sw2016.h5.88h5.cn/profile)

# Usage
```vue
<template>
	<calendar :show.sync="show" :date-text.sync="dateText" v-if="show"></calendar>
</template>
```
